# UI-Animation-
Стилі анімації: 
Linear (Лінійний) — найнудніший стиль. Об'єкт рухається з абсолютно однаковою швидкістю від початку до кінця. Має дуже роботоподібний і дешевий вигляд.

Quad / Cubic / Quint — класичні плавні стилі. Рух починається швидко, але м'яко сповільнюється біля фінішу. Ідеально підходить для появи вікон меню.

Elastic (Еластичний) — ефект гумки або пружини. Об'єкт долітає до кінця, трохи перелітає його, повертається назад і коливається. Робить інтерфейс «живим» і мультяшним.

Bounce (Стрибучий) — ефект м'ячика, який падає на підлогу. Об'єкт буквально б'ється об кінцеву точку.
